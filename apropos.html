À Propos du Site Web

Documentation technique du projet

Réalisation technique

Ce site web a été développé dans le cadre de la SAÉ S1.04 et de la ressource R1.02 Développement d'Interfaces Web. Il constitue la version web du rapport de nos travaux sur la base de données KDou.

Technologies utilisées

HTML5

Structure sémantique du document avec balises appropriées (nav, main, section, footer, article, etc.)

CSS3

Mise en forme complète avec Flexbox pour le positionnement, variables CSS pour la cohérence, et media queries pour le responsive design

Fonctionnalités implémentées

  • Navigation fixe : Barre de navigation toujours accessible en haut de page
  • Design responsive : Adaptation automatique à différentes tailles d'écran
  • Tableaux structurés : Présentation claire des résultats de requêtes
  • Code SQL mis en valeur : Coloration syntaxique pour améliorer la lisibilité
  • Cartes et boxes d'information : Organisation visuelle du contenu
  • Charte graphique cohérente : Palette de couleurs professionnelle
  • Animations subtiles : Effets de transition pour améliorer l'expérience

Architecture du projet

projet/
├── index.html              # Page d'accueil
├── mission1.html           # Mission 1 - Étude des données
├── mission2.html           # Mission 2 - Interface low code
├── mission3.html           # Mission 3 - Requêtes obligatoires
├── mission3-requetes.html  # Mission 3 - Requêtes personnalisées
├── conclusion.html         # Conclusion et recommandations
├── apropos.html           # Cette page
└── style.css              # Feuille de style unique

Contraintes respectées

  • Aucun outil WYSIWYG ou générateur de code utilisé
  • Code HTML5 valide avec balises sémantiques
  • CSS organisé et maintenable avec commentaires
  • Site responsive testé sur différents navigateurs
  • Accessibilité prise en compte (alt sur images, focus visible)
  • Navigation entre pages via liens relatifs
  • Charte graphique unique pour tout le site

Éléments HTML/CSS utilisés

Images

Utilisées avec attributs alt pour l'accessibilité

Tableaux

Structure sémantique avec thead, tbody, th, td

Listes à puces

Listes ul/ol et listes stylisées personnalisées

Flexbox

Positionnement et alignement des éléments
